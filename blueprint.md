# **App Name**: CouponConnect

## Core Features:

- Coupon Listings: Display available coupons in a grid or list format, showing key details like title, description, price, and expiration date.
- Coupon Upload: Allow sellers to upload coupon details including title, description, price, expiration date, and an image.
- Search and Filter: Enable users to search and filter coupons by category, price range, or expiration date.
- User Authentication: Implement secure user registration and login.
- Real-time Chat: Provide a real-time, one-on-one chat system for buyers and sellers to communicate.
- Coupon Verification Tool: LLM assesses uploaded coupons against terms provided in a dataset of sample valid and invalid coupon terms. This tool determines whether the coupon meets basic verification criteria, assigning a confidence score which moderators use in their assessment. It highlights potential issues for review.
- User Profiles: Simple profile pages for buyers and sellers to view their listings and chat history.

## Style Guidelines:

- Primary color: Muted blue (#6699CC) for a professional and trustworthy feel.
- Background color: Light gray (#F0F0F0) to maintain a neutral backdrop.
- Accent color: Soft green (#8FBC8F) for calls to action and visual interest.
- Body and headline font: 'PT Sans', a humanist sans-serif, offering a blend of modernity and warmth for readability and clear communication.
- Use clear and simple icons for categories and actions, ensuring they are easily understandable.
- Maintain a clean and intuitive layout, ensuring smooth navigation and a seamless user experience.
- Implement subtle animations to provide feedback and enhance interactivity.
